Bashsays v1.0.0 by Ewerton Bramos
GitHub: @KaiXtr
   bashsays [-args] [args value] "message"
   
   -o [file]: show the contents of a file.
   -i: run the message forever.
   -r: run the message from left to right.
   -s: run the message at a slower pace. Can be repeated up to 5 times.
   -f: run the message at a faster pace. Can be repeated up to 5 times.
   -q: show the message multiple times at the screen. Can be repeated up to 2 times.
   -c [1-16]: change default/first text color.
   -b [1-16]: change border color.
      1: black         9: bright black
      2: red          10: bright red
      3: green        11: bright green
      4: yellow       12: bright yellow
      5: blue         13: bright blue
      6: magenta      14: bright magenta
      7: cyan         15: bright cyan
      8: white        16: bright white
   -t [0|1+]: color blinking timeout.
   -p [text symbol] [bg symbol] [border symbol]: change message symbols.

   Exit the program with CTRL^C or Q.
